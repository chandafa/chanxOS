# Linux Basic

## Folder penting

- `/home` : folder user.
- `/etc` : konfigurasi sistem.
- `/usr` : aplikasi dan library.
- `/var` : log dan data yang sering berubah.
- `/tmp` : file sementara.

## Command dasar

```bash
pwd
ls
cd Documents
mkdir project
rm file.txt
cp file.txt backup.txt
mv old.txt new.txt
```
